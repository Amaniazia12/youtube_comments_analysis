MSP Ain shams team, youtube comments sentiment analysis api

This api is designed to get a youtube video id from the client and perform the following:
  • Collect comments on the given youtube video.
  • Perform sentiment analysis on each of the comments
  • Return to the client the ratio positive_comments/total_number_of_comments

To use the api, use a post method to the url ("https://hollow-orange.glitch.me/sentiment_analysis") and include the following parameters using url encoded:
  • num_of_comments: the maximum number of comments to collect from youtube (currently doesn't affect operation of the api)
  • video_id: the id (not the url) of the youtube video
